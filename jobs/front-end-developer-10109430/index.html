This is an excellent opportunity for someone with solid leadership skills who is looking for an opportunity learn new technologies while guiding the development of a new front-end web based application.

What does an exceptional Software Development Engineer, Staff level person do?

In this role, you will collaborate with Product Management, Marketing, Designers, Technology, and Architecture teams to develop a front-end Web Application. You will work in an iterative/Agile environment providing technical leadership for solving complex programming tasks. You’re responsible for the usage of technologies across the existing client products. Your expertise in building reusable components and integrating them with the backend services is required. You will also perform integration testing along with designing and maintaining coding standards to improve application development and maintenance. You will also assist the QA team in explaining functional requirements and reviewing test cases.

Essential Responsibilities:

  • Translate requirements and conceptual designs into high quality, visually appealing and intuitive user interfaces and user experiences
  • Take ownership of front-end components and drive continual expansion, reuse, and improvement across the board
  • Integrate with backend infrastructure
  • Analyze and solve problems across all levels of the stack
  • Produce clean, secure, testable, maintainable, reusable, well-documented code, and participate in and drive regular code reviews
  • Unit-test code for robustness, including usability and general reliability
  • Work closely with the Product Owner on product development and with the QA Team on bug fixing and improving application performance
  • Optimize applications for performance, scalability, security and user experience
  • Guide fellow colleagues on improving their skills to deliver their best work
  • Maintain up-to-date knowledge of technology standards, industry trends, new technologies, and software development standard methodologies

Basic Qualifications for consideration:

  • BS/MS degree in Computer Science, Engineering or a related subject
  • Minimum of 2+ years of experience with application development with strong design principles and user interface guidelines.
  • Demonstrated success building dynamic, responsive applications using HTML, CSS and JavaScript
  • Experience with front-end component libraries such as Bootstrap, Foundation or SemanticUI
  • Strong understanding of Graceful Degradation and Progressive Enhancement
  • Experience using modern JavaScript syntax; ES6 and beyond
  • CSS Preprocessor experience, understanding of CSS Modules and CSS-in-JS approaches a plus
  • Understanding of Flux and Unidirectional front-end architectures, Redux and Redux Middleware
  • Experience with ReactJS, VueJS or Angular
  • Front-end build processes using GulpJS or Webpack
  • Understanding of JavaScript module loaders, CommonJS and ES Harmony
  • Proficiency using data from various sources including: JSON Rest APIs and experience parsing XML
  • Single Page Application state management using browser storage and in-memory databases
  • Understanding of CORS and JSONP requests
  • Experience with promises and using APIs like Fetch, Bluebird and Axios

Travel:

  • Minimal travel 10% which may include travel to Fiserv locations for meetings/training.
